
Mettez le fichier des liens dans le méme dossier que le fichier PageRank.c et nommez le net.txt
les fichiers :
 indegrees.txt contient le degres entrant de chaque sommet non isolé et qui emmene vers d'autre noeud
 outdegrees.txt contient le degres sortant de chaque sommet non isolé et qui emmene vers d'autre noeud
 pagerank.txt contient le pagerank de chaque sommet non isolé 

compilation : gcc PageRank.c -o PageRank
execution : widows PageRank.exe --linux--./PageRank
